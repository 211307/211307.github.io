CSS 基础

  1. CSS 简介

    1. CSS 是 层叠样式表
    2. CSS 是一种标记语言
    3. CSS 主要用于设置 HTML 页面中的文本内容（字体、大小、对齐方式等）、图片的外形（宽高、边框样式、边距等）以及版面的布局和外观显示样式。
    4. CSS 美化 HTML，布局页面。

  2. CSS 格式

    • 放置位置
      • CSS 语句放在 <head></head> 范围中的 <style></style> 标签中，HTML 语句放在 <body></body> 标签中。
    • 格式：
      • 样式选择器 {属性:值;}
    • 格式介绍
      • 样式选择器：是用于指定 CSS 样式的 HTML 标签，花括号内是对该对象设置的具体样式。
      • 属性和属性值以 “键值对” 的形式出现
      • 属性是对指定的对象设置的样式属性，如：字体大小、文本颜色等。
      • 属性和属性值之间用英文 “ : ” 分开。
      • 多个 “键值对” 之间用英文 “ ; ” 进行区分。

  3. CSS 代码风格

    1. 样式书写

      • 紧凑格式：
      选择器 {color: deeppink;}
      • 展开格式：
      选择器 {
      color: pink;
}

    2. 样式大小写风格

      • 样式选择器、属性名、属性值关键字全部使用小写字母，特殊情况除外。

    3. 样式空格风格

      • 属性值前面，冒号后面，保留一个空格。
      • 选择器和大括号中间保留空格。

  4. CSS 选择器的作用

    1. 选择器作用

      • 根据不同需求把不同的标签选出来这就是选择器作用。

    2. 选择器分类

      • 选择器分为 基础选择器 和 复合选择器。

    3. CSS 基础选择器

      • 基础选择器介绍
        • 基础选择器是由 单个选择器组成。
        • 基础选择器有包括：标签选择器、类选择器、id 选择器 和 通配符选择器。
      • 4 种基础选择器

        1. 标签选择器

          • 标签选择器（元素选择器）是指用 HTML 标签名称作为选择器，按标签名称分类，为页面中某一类标签指定同一的 CSS 样式。
          标签名 {
      属性1: 属性值1;
}

        2. 类选择器

          • 如果想要差异化选择不同的标签，单独选择一个或者某几个标签，可以使用 类选择器。
          <!-- 单类名 -->
   <!-- 调用语句 -->
   <div class="类名">变红</div>

   <!-- 创建语句 -->
   .类名 {
      属性1：属性值1;
   }

<!-- 多类名 -->
<div class="red font20">亚瑟</div>
          • 在标签 class 属性中写 多个类名
          • 多个类名中间 必须 用空格分开

        3. id 选择器

          • id 选择器可以为标有特定 id 的 HTML 元素指定特定的样式。
          • HTML 元素以 id 属性来设置 id 选择器，css 中 id 选择器以 # 来定义。
          • 一般和 JavaScript 搭配使用。
          • id 选择器 只能 调用一次。
          <!-- 创建语句 -->
#id名 {
    属性1：属性值;
}

        4. 通配符选择器

          • 在 css 中，通配符选择器使用 “ * ” 定义，它表示选取页面中所有元素（标签）。
          • 通配符选择器不需要调用，自动就给所有的元素使用样式。
          * {
    属性1：属性值1;
}

    4. CSS 的复合选择器

      1. 什么是复合选择器

        1. 在 CSS 中，可以根据选择器的类型把选择器分为 基础选择器 和 复合选择器
        2. 复合选择器是由两个或多个基础选择器，通过不同的方式组合而成。
        3. 复合选择器包括：后代选择器、子选择器、并集选择器、伪类选择器等等。

      2. 后代选择器

        1. 后代选择器又称为包含选择器，可以选择父元素里面子元素。

        2. 代码：

          父元素 子元素 { 属性: 属性值;}
        • 注意：
          • 父元素 和 子元素一定要使用空格分开
          • 后代选择器只会修改子元素。
          • 后代选择器只要被父元素包含就可以一层一层往下查找。
          • 父元素和子元素可以是任意选择器。

      3. 子选择器

        • 子元素选择器只能选择作为某元素的最近一级子元素。

        • 语法：

          父元素>子元素 { 样式声明; }

        • 注意：

        • 父元素和子元素中间用 大于号 隔开。

      4. 并集选择器

        • 并集选择器可以选择多组标签，同时为他们定义相同的样式。

        • 并集选择器是各种选择器通过英文逗号（,）链接而成，任何形式的选择器都可以作为并集选择器的一部分。

        • 语法：

          元素1,元素2 { 样式声明; }

      5. 伪类选择器

        • 伪类选择器用于向某些选择器添加特殊的效果，比如给链接添加特殊效果，或选择第 1 个，第 n 个元素。
        • 伪类选择器书写最大的特点是用冒号 ( : ) 表示。
        • 伪类选择器有很多种，比如有链接伪类、结构伪类等。
        1. 链接伪类选择器
          1. 属性：
            1. a:link { 样式声明; } /修改没有被访问的链接,选择所有未被访问的链接/
            2. a:visited { 样式声明; } /修改被点击过的链接，选择所有已被访问的链接/
            3. a:hover { 样式声明; } /选择鼠标指针位于其上的链接/
            4. a:active { 样式声明; } /选择活动链接 （鼠标按下未松开的链接）/
          2. 注意：
            1. 为了确保生效，请按照 LVHA 的顺序声明：link -: visited -: hover -: active。
            2. 因为链接有单独的格式，所以在实际工作中需要单独指定样式。
        2. :focus 伪类选择器

          • :focus 伪类选择器 用于选取过得焦点的表单元素。

          • 焦点就是光标，一般情况 <input> 类表单元素才能获取。

          • 格式：

            input:focus { 属性: 属性值; }

  5. CSS 字体属性

    • CSS Fonts（字体）属性用于定义 字体系列、大小、粗细、和字体样式（如斜体）。
    1. 字体系列

      1. 设置字体系列

        • CSS 使用 font-family 属性定义文本的字体系列。
        • 各个字体之间必须使用英文状态下的逗号隔开。
        • 一般情况下，如果有空格隔开的多个单词组成的字体，加引号。（多字体使用来防止客户没有第一个字体）
        div {
    font-family: Arial,"Microsoft Yahei","微软雅黑";
}

      2. 字体大小

        • css 使用 font-size 属性定义字体大小。
        • px（像素）大小是我们网页的最常用的单位。
        选择器 {
    font-size: 20px;       <!-- 数字控制 -->
}

      3. 字体粗细

        • css 使用 font-weight 属性定义字体粗细。
        选择器 {
    font-weight: 属性值;
    <!-- 建议使用数字控制 属性值：400 或者 normal 不加粗，700 或 bold 加粗 -->
}

      4. 文字样式

        • CSS 使用 font-style 属性设置文本的风格。
        选择器 {
    font-style：normal;
    <!-- normal：默认值。italic：浏览器会显示斜体文字。 -->
}

      5. 字体复合属性

        • 复合属性：简写的方式，节约代码

        • 格式：

          选择器 {
    font: font-style font-weight font-size/line-height font-family;
}

        • 注意：

          • 使用 font 属性时，必须按上面语法格式中的顺序书写，不能更换顺序，并且各个属性间以空格隔开。
          • 复合属性值在对应的位置输入就行，font-style 这个不用写。
          • font-size 和 font-family 属性必须保留，其它属性可以不用写。

  6. 文本属性

    • CSS Text（文本）属性可定义文本的 外观，比如文本的颜色、对齐文本、装饰文本、文本缩进、行间距等。

    1. 文本颜色

      • color 属性用于定义文本的颜色。
      选择器 {
   color: red;
}
      表示 属性值
      预定义的颜色值 red，green，blue，pink
      十六进制 #FF0000，#FF6600，#29D794
      RGB代码 rgb(255,0,0)或rgb(100%,0%,0%)
      • 开发中用的最多的是十六进制。

    2. 对齐文本

      • text-align 属性用于设置元素内文本内容的 水平对齐 方式。
      选择器名 {
   text-align: center;
}
      属性值 解释
      left 左对齐
      right 右对齐
      center 居中对齐

    3. 装饰文本线

      • text-decoration 属性规定添加到文本的修饰。可以给文本添加下划线、删除线、上划线等。
      选择器名 {
    text-decoration: underline;
}
      属性值 描述
      none 没有装饰线
      underline 下划线
      overline 上划线
      line-through 删除线

    4. 文本缩进

      • text-indent 属性用来指定文本的第一行的缩进，通常是 段落的首行缩减。
      p {
   text-indent: 20px; <!-- 通过数值控制缩进距离 -->
   text-indent:2em;  
   <!-- em 是一个相对单位，根据当前文字大小调整缩进距离。否则会按照父元素调整 -->
}

    5. 行间距

      • line-height 属性用于设置行间的距离（行高）。

      • 行高：上间距、文本高度、下间距。

        p {
  line-height: 20px;
}

  7. CSS 引入方式

    • CSS 引入概述
      • 按照 CSS 样式书写的位置（或者引入的方式），CSS 样式表可以分为三大类：
        • 内部样式表（嵌入式）
        • 行内样式表（行内式）
        • 外部样式表（连接式）
    • CSS 三种样式表

      1. 内部样式表

        • 内部样式表（内嵌样式表）是把所有的 CSS 代码 放入到 HTML 页面中的 <style> 标签范围中。

        • <style> 标签理论上可以放在 HTML 文档的任何地方，推荐放在文档的 <head> 标签中。

        • 方便控制当前整个页面中的元素样式设置。

        • 代码结构清晰，但没有实现结构与样式完全分类，不利于 CSS 代码的重复使用。

          <style>
    选择器名 {
        属性: 属性值;
    }
</style>

      2. 行内样式表

        • 行内样式表（内联样式表）是在元素标签内部使用 style 属性中设定 CSS 样式。
        • 适合简单修改样式。
        <p style="属性1:属性值1; 属性2: 属性值2;"></p>
        • style 其实就是标签的属性。
        • 在双引号中间，写法要符合 CSS 规范。
        • 可以控制当前的标签设置样式。

      3. 外部样式表

        • 实际开发都是外部样式表，适合于样式比较多的情况。

        • 核心：样式单独写到 CSS 文件中，之后把 CSS 文件引入到 HTML 页面中使用。

        • 在 HTML 页面中，使用 <link> 标签引入这个文件。

          <head>
    <link rel="stylesheet" href="CSS 文件路径">
</head>

        • 属性

          • rel：定义当前文档与被链接文档之间的关系，在这里需要指定为 “stylesheet”，表示被链接的文档是一个样式表文件。
          • href：定义所链接外部样式表文件的 URL，可以是相对路径，也可以是绝对路径。

  8. 浏览器调试工具

    1. 打开浏览器按 f12 或者 右击页面选择检查元素。
    2. 使用调试工具
      1. Ctrl + 滚轮 可以放大开发者工具代码大小。
      2. 左边是 HTML 元素结构，右边是 CSS 样式。
      3. 右边 CSS 样式可以改动数值（左右箭头或者直接输入）和查看颜色。
      4. Ctrl + 0 复原浏览器大小。
      5. 如果右击元素，发现右侧没有样式引入，极有可能是类名或者样式引入错误。
      6. 如果有样式，但是样式前面有 黄色叹号提示，则是样式属性书写错误。
    3. 调试工具只能检查代码，不能修改代码。

入门

  1. Emmet 语法

    • Emmet 语法概述

      • Emmet 语法的前身是 Zen coding，它使用缩写，来提高 HTML/CSS 的编写速度，Vscode 内部已经集成该语法。

    • 使用

      1. 快速生成 HTML 标签
        1. 生成标签，直接输入标签 按 tab 键或 Enter 键即可。
        2. 生成多个标签的格式：标签名*数值 按 回车 或 tab 键。
        3. 生成父子级标签格式：父标签名>子标签名 按 回车键。
        4. 生成兄弟级标签格式：标签名+标签名 按 回车键。
        5. 生成带有类的标签格式：标签名.元素值 按 回车键。
        6. 生成带有id的标签格式：标签名#id值 按 回车。
        7. 生成标签类名是有顺序的，可以用 自增符号 $：标签名.元素值$*数值
        8. 在生成的标签内部写内容可以用 { } 表示：标签名{内容}
      2. 快速生成 CSS 样式语法
        1. 输入属性时 输入单词的首个字母然后回车。
      3. 快速格式化代码
        1. 在代码页右击选择 格式化文档。

  2. CSS 的元素显示模式

    1. 元素显示模型特点

      1. 元素显示模式就是元素（标签）以什么方式进行显示。
      2. 元素显示模式的分类
        1. HTML 元素一般分为块元素和行内元素两种类型。

    2. 块元素

      1. 常见的快元素有 <h1>~<h6>、<p>、<div>、<ul>、<ol>、<li> 等，其中 <div> 标签是最典型的块元素。
      2. 块元素特点：
        1. 比较霸道，自己独占一行。
        2. 高度、宽度、外边距以及内边距都可以控制。
        3. 宽度默认是容器（父级宽度）的 100%。
        4. 是一个容器及盒子，里面可以放行内或者块级元素。
      3. 注意：
        1. 文字类标签内不能使用块级元素。如：<p> 或 <h1>~<h6> 标签内不能使用块级元素。
        2. 让文字的行高等于块元素的高度，就可以让文字垂直居中。
          1. 垂直居中的原理：就是让上间距和下间距把文字挤到中间。

    3. 行内元素

      1. 常见的行内元素有 <a>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<span> 等，其中 <span> 标签是最典型的行内元素。有的地方也将行内元素称为内联元素。
      2. 行内元素特点：
        1. 相邻行内元素在一行上，一行可以显示多个。
        2. 高、宽直接设置是无效的。
        3. 默认高度就是它本身内容的高度。
        4. 行内元素只能容纳文本或其他行内元素。
      3. 注意：
        1. 链接里面不能再放链接。
        2. 特殊情况链接 <a> 里面可以放块级元素，但是给 <a> 转换一下块级模式最安全。

    4. 行内块元素

      1. 在行内块元素中有几个特殊的标签 —— <img/>、<input />、<td>，它们同时具有块元素和行内元素的特点。
      2. 行内块元素特点：
        1. 和相邻行内元素（行内块）在一行上，但是它们之间会有空白缝隙。一行可以显示多个（行内元素特点）。
        2. 默认宽度就是它本身内容的宽度（行内元素特点）。
        3. 高度、行高、外边距以及内边距都可以控制（块级元素特点）。

    5. 元素显示模式转换

      1. 行内元素转换成块级元素

        1. 例题：

          行内标签名 {
   display: block; /*转换格式*/
}

      2. 块级元素转换成行内元素

        1. 例题：

          块级元素 {
   display: inline; /*转换语句*/
}

      3. 转换成行内块元素：display: inline-block;

  3. CSS 的背景

    1. 特点

      • 背景属性可以设置背景颜色、背景图片、背景平铺、背景图片位置、背景图像固定等。
      • 实际开发常见于 logo 或者一些装饰性的小图片或者是超大的背景图片，优点是非常便于控制位置。（CSS 精灵图也是一种运用场景）

    2. 背景颜色

      1. background-color 属性定义了元素的背景颜色。

      2. 格式：

        background-color:颜色值;

    3. 背景图片

      1. background-image 属性描述了元素的背景图像。

      2. 代码

        background-image : none | url("url 地址");
/* none：无背景图（默认的）
   url：使用绝对或相对地址指定背景图像
   使用 div 标签作为 选择器的时候需要设置 高 和 宽
   */

    4. 背景平铺

      1. background-repeat 属性对图片进行平铺设置

      2. 代码：

        background-repeat: repeat | no-repeat | repeat-x | repeat-y;
/*
repeat：平铺图片
no-repeat：不平铺图片
repeat-x：沿着横向平铺
repeat-y：沿着纵向平铺
*/

    5. 背景图片位置

      • 利用 background-position 属性可以改变图片在背景中的位置。
      background-position: x y;
<!-- x 轴 左右，y 轴 上下 -->
      • 参数代表的意思是：x 坐标和 y 坐标。可以使用 方位名词 或者 精确单位。
        • 参数是方位名词：
          • 如果指定的两个值都是方位名词，则和两个值前后顺序无关，比如 left top 和 top left 效果一致。
          • 方位名词参数可以省略一个。
            • 省略 x 轴，x 轴使用默认的参数：水平居中显示。
            • 省略 y 轴，y 轴使用默认的参数：垂直居中显示。
        • 参数是精确单位：
          • 如果参数值是精确坐标，那么第一个肯定是 x 坐标，第二个一定是 y 坐标。
          • 如果只指定一个数值，那该数值一定是 x 坐标，另一个默认垂直居中。
        • 参数是混合单位：
          • 如果指定的两个值是精确单位和方位名词混合使用，则第一个值是 x 坐标，第二个值是 y 坐标。
        参数值 说明
        length 像素 px | 百分数 | 由浮点数字和单位标识符组成的长度值是 精确单位
        position top | center | bottom | left | center | right 方位名词

    6. 背景图像固定（背景附着）

      • background-attachment 属性设置背景图像是否 固定 或者随着页面的其余部分 滚动。

      • background-attachment 后期可以制作视差滚动的效果。

        background-attachment: fixed | scroll;
        参数 作用
        scroll 背景图像是随对象内容滚动（默认是滚动的）
        fixed 背景图像固定

    7. 背景属性复合写法

      • 为了简化背景属性的代码，我们可以将这些属性合并简写在同一个属性 background 中。从而节约代码量。
      • 当使用简写属性时，没有特定的书写顺序，一般习惯约定顺序为：
        • background: 背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置;
      • 注意：
        • 各个属性之间必须使用空格隔开。

    8. 背景色半透明

      • CSS3 为我们提供了背景颜色半透明的效果。

        background: rgba(0, 0, 0, 0.3);
<!-- 
r：红色。
g：绿色。
b：蓝色。
a：alpha 透明度，取值范围在 0~1 之间，可以把 0.3 的 0 省略掉，写成 background: rgba(0, 0, 0, .3);
 -->

      • 注意：

        • 背景半透明是指盒子背景半透明，盒子里面的内容不受影响。

  4. CSS 的三大特性

    • 概述：
      • CSS 有三个非常重要的三个特性：层叠性、继承性、优先级。

    1. 层叠性

      • 概述：
        • 相同选择器给设置相同的样式，此时一个样式就会**覆盖（层叠）**另一个冲突的样式。层叠性主要解决样式冲突的问题
      • 层叠性原则：
        • 样式冲突，遵循的原则是就近原则，那个样式离结构近，就执行那个样式
        • 样式不冲突，不会层叠。

    2. 继承性

      • 概述：
        • CSS 中的继承：子标签会继承父标签的某些样式，如文本颜色和字号。简单的理解就是：子承父业。
        • 恰当地使用继承可以简化代码，降低 CSS 样式的复杂性。
        • 子元素可以继承父元素的样式（tect-，font-，line- 这些元素开头的可以继承，以及 color 属性）
      • 行高的继承
        • 行高可以直接在字体大小后面添加。
        • 格式：font: 12px/24px '宋体';
        • 注意：
          • 行高可以跟单位也可以不跟单位。
          • font: 12px/1.5 '宋体';：
            • 不添加单位的意思就是：子元素的行高 = 当前字体大小 * 1.5
          • 这样写法最大的优势就是里面子元素可以根据自己文字大小自动调整行高

    3. 优先级

      • 概述

        • 当同一个元素指定多个选择器，就会有优先级的产生。

      • 优先级特性

        • 选择器相同，则执行层叠性。
        • 选择器不同，则根据选择器权重执行。

      • 选择器权重如下

        选择器 选择器权重
        继承 或者 * 0,0,0,0
        元素选择器（标签选择器） 0,0,0,1
        类选择器，伪类选择器 0,0,1,0
        ID 选择器 0,1,0,0
        行内样式 style=" " 1,0,0,0
        !important 重要的 ∞ 无穷大
        • !important 使用格式：
          • color:slateblue !important;
          • 注意：
            • !important 直接添加到 属性值 后面。
            • 如果有多个属性值添加 !important 将会比较这俩选择器的优先级。

      • 优先级注意事项

        1. 权重是有 4 组数字组成，但是不会有进位。
        2. 等级判断从 选择器权重 的左向右比较，如果某一位数值相同，则判断下一位数值。
        3. 简单记忆法：通配符和继承权重为 0，标签选择器为 1，类（伪类）选择器为 10，id 选择器为 100，行内样式表为 1000，!important 无穷大。
        4. 继承的权重为0，如果该元素没有直接选中，不管父元素权重多高，子元素得到的权重都是 0。
        5. a 链接 浏览器默认制定了一个样式 蓝色的 有下滑划线，可以通过层叠性修改。

      • 权重叠加

        • 特点
          • 如果是复合选择器，则会有权重叠加，需要计算权重。
          • 权重叠加就是：两个权限数值相加。
            • 如：ui li {} 权限等于 0,0,0,1 + 0,0,0,1 = 0,0,0,2
          • 如果 标签选择器 权限叠加超过 9 ，那么呢个权限是 0,0,0,10

  5. CSS 盒子模型

    • 概述
      • 页面布局要学习三大核心，盒子模型，浮动 和 定位。
    • 网页布局的本质
      • 网页布局过程
        1. 先准备好相关的网页元素，网页元素基本都是盒子 Box。
        2. 利用 CSS 设置好盒子样式，然后摆放到相应位置。
        3. 往盒子里面装内容。
      • 网页布局的核心本质：就是利用 CSS 摆盒子。
    • 盒子模型（Box Model）组成
      • 概述
        • 盒子模型：CSS 盒子模型本质上是一个盒子，封装周围的 HTML 元素，它包括：边框、外边框、内边框、和 实际内容。
      • 边框（border）
        • 概述：
          • border 可以设置元素的边框。边框有三部分组成：边框宽度（粗细）、边框样式（实线虚线）、边框颜色。
        • 边框基础设置

          • 语法：

            <!-- 复合写法 -->
border: border-width || border-style || border-color; <!-- 没有顺序 -->
            • 参数：
              • border-width：定义边框粗细，单位是px
              • border-style：边框的样式
                • 边框的属性值：solid 实线边框、dashed 虚线边框、dotted 点线边框、none 无边框
              • border-color：边框的颜色，属性值：transparent;
        • 边框分开设置

          • 代码：

            <!-- 边框分开写法 -->
border-top: border-width || border-style || border-color; <!-- 只设定上边框，其余同理 -->
            • border 四条边设置：
              • border-top：只设置上边框。
              • border-bottom：只设置下边框。
              • border-left：只设置左边框
              • border-right 右设置右边框
        • 表格的细线边框

          • 概述

            • border-collapse 属性控制浏览器绘制表格边框的方式。它控制相邻单元格的边框。

          • 语法：

            border-collapse: collapse;

          • 参数

            • collapse 单词是合并的意思。
            • border-collapse: collapse; 表示相邻边框合并在一起。
        • 边框会影响盒子实际大小
          • 解决方案：
            1. 测量盒子大小的时候，不量边框。
            2. 如果测量的时候包含了边框，则需要 width/height 减去边框宽度。
      • 内边框（padding）
        • 概述：
          • padding 属性用于设置内边距，即边框与内容之间的距离。
        • padding 的属性
          • padding-left：左内边距。
          • padding-right：右内边距。
          • padding-top：上内边距。
          • padding-bottom：下内边距。
        • padding 复合写法
          • 概述：
            • padding 属性（简写属性）可以有一到四个值。
          • 属性：
            • padding: 5px; ：1 个值，代表上下左右都有 5 像素内边距；
            • padding: 5px 10px; ：2 个值，代表上下内边距 5 像素，左右内边距是 10 像素；
            • padding：5px 10px 20px; ：3 个值，代表上内边距 5 像素，左右内边距 10 像素，下内边距 20 像素；
            • padding：5px 10px 20px 30px; ：4个值，上是 5 像素，右 10 像素，下 20 像素，左 30 像素 顺时针。
          • 注意：
            • 如果保证盒子跟效果图大小保持一致，则让 width/height 减去多出来的内边距大小 即可。
            • 如果没有给盒子（如：div）设置 高度 或 宽度 ，那么 padding 不会撑开盒子。
      • 外边框（margin）
        • 概述：
          • margin 属性用于设置外边距，即控制盒子和盒子之间的距离。
        • margin 属性：
          • margin-left：左外边距
          • margin-right：右外边距
          • margin-top：上外边距
          • margin-bottom：下外边距
        • margin 复合写法
          • 格式：
            • margin 属性（简写属性）可以有一到四个值。
          • 属性：
            • margin: 5px; ：1 个值，代表上下左右都有 5 像素内边距；
            • margin: 5px 10px; ：2 个值，代表上下内边距 5 像素，左右内边距是 10 像素；
            • margin：5px 10px 20px; ：3 个值，代表上内边距 5 像素，左右内边距 10 像素，下内边距 20 像素；
            • margin：5px 10px 20px 30px; ：4个值，上是 5 像素，右 10 像素，下 20 像素，左 30 像素 顺时针。
        • 外边距应用
          • 外边距可以让块级盒子 水平居中，但是必须满足两个条件：

            • 盒子必须指定了宽度（width）。

            • 盒子 左右的外边距 都设置为 auto。

            • 代码：

              .header {width: 960px; margin: 0 auto;}

            • 常见的写法，以下三种都可以：

              • margin-left：auto; margin-right：auto;
              • margin：auto;
              • margin：0 auto;

            • 注意

              • 以上方法时让块级元素水平居中，**行内元素或者行内块元素水平居中给其父元素添加 text-align:center 即可。
        • 嵌套块元素垂直外边距的塌陷
          • 概述：
            • 对于两个嵌套关系（父子关系）的快元素，父元素有上外边距同时子元素也有上外边距，此时父元素会塌陷较大的外边距值。
          • 解决方案：
            • 可以为父元素定义上边框。
            • 可以为父元素定义上内边距。
            • 可以为父元素添加 overflow: hidden;
      • 清除内外边距

        • 概述：

          • 网页元素很多都带有默认的内外边距，而且不同浏览器默认的也不一致。因此我们在布局前，首先要清除下网页元素的内外边距。

        • 代码：

          * {
      padding:0; /* 清除内边距 */
      margin:0;  /* 清除外边距 */
}

        • 注意：

          • 行内元素（汉字）为了照顾兼容性，尽量只设置左右内外边距，不要设置上下内外边距。但是转换为块级和行内块元素就可以了。
      • 圆角边框

        • border-radius 属性用于设置元素的外边框圆角。

        • 语法：

          border-radius：length;  /* 数值可以是 百分比*/
border-radius：10px 20px 30px 40px; /* 四个值用来修改正方形的四个角*/
border-top-left-radius: 20px /* 单独修改左上角*/

        • radius 半径（圆的半径）原理：（椭）圆与边框的交集形成圆角效果。

        • 圆的制作：

          • 圆形做法：把数值设置为高度和宽度的一半。
          • 圆角矩形做法：把数值设置为高度的一半。
      • 盒子阴影

        • box-shadow 属性为盒子添加阴影。

        • 语法：

          box-shadow: h-shadow v-shadow blur spread color inser; （使用 1px）
h-shadow：必需。水平阴影的位置。允许负值。
v-shadow：必需。垂直阴影的位置。允许负值。
blur：可选。模糊距离。
spread：可选。阴影的尺寸。
color：可选。阴影的颜色。
inset：可选。将外部阴影（outset）改为内部阴影。

        • 注意：

          • 默认的是外阴影（outset），但是不可以写这个单词，否则导致阴影无效。
          • 盒子阴影不占用空间，不会影响其他盒子排列。
          • 可以用 hover 让鼠标经过盒子出现阴影。
      • 文字阴影

        • text-shadow 属性将阴影应用于文本。

        • 语法：

          text-shadow：h-shadow v-shadow blur color;

h-shadow：必须。水平阴影的位置。允许负值。
v-shadow：必须。垂直阴影的位置。允许负值。
blur：可选。模糊的距离。
color：可选。阴影的颜色。

  6. 浮动

    • 传统网页布局的三种方式
      • 网页布局第一准则：多个块级元素纵向排列找标准流，多个块级元素横向排列找浮动。
      • 网页布局第二准则：先设置盒子大小，之后设置盒子的位置。
      • 普通流（标准流）
        • 就是标签按照约定好默认方式排列。
        • 标准流就是 块级元素 和 行内块元素。
      • 浮动
      • 定位
    • 浮动特性：
      1. 浮动元素会脱离标准流
        • 浮动的盒子不再保留原先的位置。
      2. 浮动的元素会一行内显示并且元素顶部对齐。
      3. 浮动的元素会具有行内块元素的特性。
        • 如果块级盒子没有设置宽度，默认宽度和父级一样宽，但是添加浮动后，它的大小根据内容来决定。
        • 浮动的盒子中间是没有缝隙的，是紧挨着一起的。
        • 行内元素同理。
    • 浮动

      • float 属性用于创建浮动框，将其移动到一边，直到左边缘或右边缘触及包含块或另一个浮动框的边缘。

      • 语法：

        选择器 { float: 属性值; }

      • 属性值：

        • none：元素不浮动（默认值）。
        • left：元素向左浮动。
        • right：元素向右浮动。

      • 使用：

        • 可以先使用标准流的父元素排列上下位置，之后内部子元素采取浮动排列左右位置。

      • 注意：

      • 一个元素浮动，其它浮动元素也会跟着浮动。

      • 浮动的盒子只会影响浮动盒子后面的标准流，不会影响前面的标准流。

    • 清除浮动

      • 概述：

        • 清除浮动的本质是清除浮动元素造成的影响。
        • 如果父盒子本身有高度，则不需要清除浮动。
        • 清除浮动之后，父级就会根据浮动的子盒子自动检测高度。父级有了高度，就不会影响下面的标准流了。

      • 语法：

        选择器 {clear: 属性值;}

left：不允许左侧有浮动元素（清除左侧浮动的影响）
right：不允许右侧有浮动元素（清除右侧浮动的影响）
both：同时清除左右两侧浮动的影响。

      • 清除浮动方法：

        1. 额外标签法 也称为隔墙法，是 W3C 推荐的做法。

          • 额外标签法就是在最后一个浮动元素的后面添加一个空标签。如：</div>
            • 添加的标签必须是块级元素，不能是行内元素。
          • 优点：通俗易懂，书写方便。
          • 缺点：添加许多无意义的标签，结构化较差。

        2. 父级添加 overflow 属性。

          • 可以给父元素添加 overflow 属性，将其属性值设置为 hidden、auto 或 scroll。
          • 优点：代码简洁。
          • 缺点：无法显示溢出的部分。

        3. 父级添加 after 伪元素。

          • 代码：

            .clearfix:after{
   content: "";
   display: block;
   height: 0;
   clear: both;
   visibility: hidden;
}

.clearfix {/* IE6、7 专有*/
   *zoom: 1;
}

          • 优点：没有增加标签，结构更简单。

          • 缺点：照顾低版本浏览器。

        4. 父级添加双伪元素。

          • 代码：

            .clearfix: before,
.clearfix: after {
   content: "";
   display: table;
}
.clearfix: after {
   clear: both;
}
.clearfix {
   *zoom: 1;
}

          • 优点：代码更简洁。

          • 缺点：照顾低版本浏览器。
