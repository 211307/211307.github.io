String 类

  1. String 类：

    1. String 概述：
      • String 类在 java.lang 包下，不需要导包。
    2. String 代表字符串：
      • java 程序中所有的双引号字符串，都是 String 类的对象。
    3. 字符串的特点：
      1. 在字符串中加号表示拼接。
      2. 字符串不可变，它们的值在创建后不能被更改。
      3. 虽然 String 的值不可变的，但是它们可以被共享。
      4. 字符串的底层原理是字节数组（byte )

  2. String 构造方法：

    1. String 构造方法格式：
      方法格式 说明
      public String() 创建一个空白字符串对象，不含有任何内容
      public String(char[] chs) 根据字符数组的内容，来创建字符串对象
      pubic String(byte[] bys) 根据字节数组的内容，来创建字符串对象
      String s = “abc”; 直接赋值的方式创建字符串对象
    2. 调用：
      1. 建议直接使用赋值的方式。
      2. 除了直接赋值的，其它都是通过 new 来调用的。

  3. String 对象的特点：

    1. 通过 new 创建的字符串对象，每一次 new 都会申请一个内存空间，虽然 内容相同，但是 地址值不同。
    2. 以 " “ 方式给出的字符串，只要 字符序列相同（顺序和大小写），地址值就相同。
    • JVM 会创建一个 String 对象，并在字符串池中对它们进行链接。

  4. 字符串常用方法

    1. 使用 == 作比较

      • 基本类型：比较的是 数据值 是否相同。
      • 引用类型：比较的是 地址值 是否相同。

    2. 使用 equals() 方法来比较字符串的内容是否相同。

      • 格式：

        System.out.println(字符串1.equals(字符串2));

        a

    3. 使用 charAt() 方法获取字符串中的字符

      • 格式：

        System.out.println(字符串名.charAT(下标));     // 字符串下标从 0 开始
System.out.println(字符串名.charAT(下标)-1);   // 获取字符串最后一个元素

    4. 使用 length() 方法获取字符串的长度：

    • 格式：

      System.out.println(字符串名.length());

StringBuilder 类

  1. StringBuilder类

    1. StringBuilder 概述

      • StringBuilder 类是一个可变的字符串类。
      • String 和 StringBuilder 的区别
        • String：内容不可变。
        • StringBuilder：内容可变。

    2. StringBuilder 类的构造方法

      • public StringBuilder()：是一个空白可变字符串对象，不含有内容。
      • public StringBuilder(String str)：根据字符串内容，创建可变字符串对象。
      • 通过对象调用。

    3. StringBuilder 的添加和反转方法

      1. public StringBuilder append(任意元素)：添加数据，并返回对象本身。

        • 格式：
        字符串对象.append(元素);
        • append 可以进行链式编程 ：
        字符串对象.append(元素).append(元素2).[...];

      2. public StringBuilder reverse()：返回相反的字符序列。

        1. 格式：

           字符串对象.reverse();

      3. 反转升级版：

        • 格式：

          // 通过方法
public static String 方法名(String 参数名){
     // 相当于匿名类
     return new StringBuilder(参数名).reverse().toString();
}

    4. StringBuilder 和 String 相互转换

      1. StringBuilder 转换为 String
        • public String toString()：通过 toString 就可以实现把 StringBuilder 转换为 String
        • 格式：
      // 创建 StringBuilder 对象
StringBuilder 对象名1 = new StringBuilder();

// 转换格式
String 对象名2 = 对象名1.toString();
      1. String 转换为 StringBuilder
        • pubic StringBuilder(String s)：通过构造方法转换。
        • 格式：
      // 转换格式：
StringBuilder 对象名2 = new StringBuilder(对象名1);

集合类

  1. 集合基础

    1. 集合概述
      • 集合类的特点：提供一种存储空间可变的存储模型，存储的数据容量可以放声改变。

  2. ArrayList 类

    1. ArrayList<E>

      • 可调整大小的数组实现。
      • <E> 是一种特殊的数据类型，泛型。

    2. ArrayList 构造方法和添加方法

      方法名 说明
      public ArrayList() 创建一个空的集合对象
      public boolean add(E e) 将指定的元素追加到此集合的末尾
      public void add(int index,E element) 在此集合中的指定位置插入指定的元素

      • ArrayList 集合调用：

        // 导包
import java.util.ArrayList;

// 创建 ArrayList new 对象
ArrayList<String> 对象名 = new ArrayList<>();     // jdk 7 以后的 ArrayList 第二个泛型可以不写。

// 添加元素
对象名.add("hello"); // 逐一添加
对象名.add(1,"world"); // 指定位置添加

    3. ArrayList 集合常用方法

      方法名 说明
      public boolean remove(Object o) 删除指定的元素，返回删除是否成功
      public E remove(int index) 删除指定索引处的元素，返回被删除的元素
      public E set(int index,E element) 修改指定索引处的元素，返回被修改的元素
      public E get(int index) 返回指定索引处的元素
      public int size() 返回集合中的元素的个数

      • ArrayList 常用方法

        // 导包
import java.util.ArrayList;

// 创建 ArrayList new 对象
ArrayList<String> 对象名 = new ArrayList<>();

// 方法的使用:

//删除元素
对象名.remove("元素");

//根据索引删除元素
对象名.remove(1);

// 根据索引修改数据
对象名.set(1,"元素");

// 返回索引的元素
对象名.get(0);

// 返回集合的个数
对象名.size();

    4. 集合遍历

      • 格式：

        // 导包
import java.util.ArrayList;

// new ArrayList类
ArrayList<String> arr= new ArrayList<>();

// 添加元素
arr.add("aaa");
arr.add("bbb");

// 循环遍历
for(int i=0;i<arr.size();i++){
     String s  = arr.get(i);
     System.out.println(s);
}

Math&System类

  1. Math 类

    1. Math 概述：
      1. Math：在 java.lang 类下不需要导包。
      2. Math包含执行基本数字运算的方法，如基本指数，对数，平方根和三角函数。
    2. Math 类的常用方法
    方法名 说明
    public static int abs(int a) 返回参数的绝对值
    public static double ceil(double a) 返回大于或等于参数的最小 double 值，等于一个整数
    public static double floor(double a) 返回小于或等于参数的最大 double 值，等于一个整数
    public static int round(float a) 按照四舍五入返回最接近参数的 int
    public static int max(int a,int b) 返回两个 int 值中的较大值
    public static int min(int a,int b) 返回两个 int 值中的较小者
    public static double pow(double a,double b) 返回 a 的 b 次幂的值
    public static double random() 返回值为 double 的正值，[0.0,1.0)
    • 使用格式：Math.方法名([参数])
    • 把 random 强转：System.out.println((int)(Math.random()*10)+1);

  2. System 类

    1. 常用方法
      方法名 说明
      public static void exit(int status) 终止当前运行的 java 虚拟机，非零表示异常终止
      public static long currentTimeMillis() 返回当前时间（以毫秒为单位）
    2. 格式：

    • 常用的操作之一：用于基本数据类型与字符串之间的转换。

      short Short
      int Integer
      long Long
      float Float
      double Double
      char Character
      boolean Boolean

  2. Integer 类的概述和使用

  5. 自动装箱和拆箱

SimpleDateFormat 类

  1. 概述：SimpleDateFormat是一个具体的类，用于以区域设置敏感的方式格式化和解析日期。

    • 日期和时间格式由日期和时间模式字符串指定，在日期和时间模式字符串中，从 ‘A’ 到 ‘Z’ 以及从 ‘a’ 到 ‘z’ 引号的字母被解释为表示日期和时间字符串的组件的模式字母。
    • 常用的模式字母以及对应关系如下；
      • y 年
      • M 月
      • d 日
      • H 时
      • m 分
      • s 秒

  2. SimpleDateFormat 的构造方法

    方法名 说明
    public SimpleDateFormat​() 构造一个 SimpleDateFormat，使用默认模式和日期格式
    public SimpleDateFormat​(String pattern) 构造一个 SimpleDateFormat 使用给定的模式和默认的日期格式

  3. SimpleDateFormat 格式化和解析日期

    1. 格式化（从 Date 到 String）

      • public final String format(Date date)：将日期格式化成日期/时间字符串

      • 代码：

        // 导包
import java.text.SimpleDateFormat;
import java.util.Date;

Date b1 = new Date();
SimpleDateFormat sbt =new SimpleDateFormat("yyyy年MM月dd日 HH:mm:ss");
String s = sbt.format(b1);
System.out.println(s);

    2. 解析（从 String 到 Date）

      • public Date parse (String source)：从给定字符串的开始解析文本以生成日期

      • 代码：

        // 导包
import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;


// 语句
String ss ="2020-08-09 11:11:15";
SimpleDateFormat sbt =new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
Date dd = sbt.parse(ss);
System.out.println(dd);

  4. 日期工具类

    1. 定义一个方法，用于把日期转换为指定格式的字符串。

      • 返回值类型：String。

      • 参数：Date date，String format。

      • 代码：

        // 导包
import java.util.Date;
// 创建 Date 类
Date b1 = new Date();
// 调用方法
String b2= Main.zf(b1,"yyyy年MM月dd日 HH:mm:ss");
System.out.println(b2);

// 导包
import java.text.SimpleDateFormat;
import java.util.Date;
// 工具类
public class Main {
   private Main(){}
   public static String zf(Date s,String  format){
      SimpleDateFormat a = new SimpleDateFormat(format);
      String bb = a.format(s);
      return bb;
   }
}

    2. 定义一个方法，用于字符串解析为指定格式的日期。

      • 返回值类型：Date。

      • 参数：String s, String format。

      • 代码：

        // 导包
import java.text.ParseException;
import java.util.Date;

// 调用方法
String a = "2019-08-19 18:10:15";
Date ss=Main.d(a,"yyyy-mm");
System.out.println(ss);

// 工具类
import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;
public class Main {
   private Main(){}
   public static Date d(String ss,String format) throws ParseException {
      SimpleDateFormat aa = new SimpleDateFormat(format);
      Date ab=aa.parse(ss);
      return ab;
   }
}

Calendar 类

  1. Calendar 类概述

    • Calendar 为某一时刻和一组日历字段之间的转换提供了一些方法，并为操作日历字段提供了一些方法。

    • Calendar 提供了一个类方法 getlnsance 用于获取 Calendar 对象，其日历字段已使用当前日期和时间初始化：

      Calendar c = Calendar.getInstance(); // 通过多态获取对象
int year = c.get(Calendar.YEAR); //年
int month = c.get(Calendar.MONTH)+1;   //月，因为它的月份从 0 开始，所以要 +1。
int date =c.get(Calendar.DATE);  // 日
System.out.println(year + "年" + month + "月" + date + "日");

  2. Calendar 的常用方法

    方法名 说明
    public int get(int field) 返回给定日历字段的值
    public abstract void add(int field,int amount) 根据日历的规则，将指定的时间量添加或减去给定的日历字段
    public final void set(int year,int month,int date) 设置当前日历的年月日

    • 方法的使用：

      Calendar c = Calendar.getInstance(); // 通过多态获取对象
// add 的使用
c.add(Calendar.YEAR,-3);   // 年 - 3。
// set 的使用
c.set(2020,8,11);    // 月从 0 开始，所以月是 9。
int year = c.get(Calendar.YEAR); //年
int month = c.get(Calendar.MONTH)+1;   //月
int date =c.get(Calendar.DATE);  // 日
System.out.println(year + "年" + month + "月" + date + "日");
